### 技术细节

- **框架版本**: React 18.3.1
- **构建工具**: Vite 5.3.1
- **样式方案**: Tailwind CSS 3.4.4
- **路由管理**: React Router 6.26.0
- **图标库**: Bootstrap Icons 1.11.3
- **Node 版本要求**: >=18.0.0

### 设计规范

- C 端采用橙色主题（原型为粉色）
- B 端采用蓝色主题
- 遵循微信小程序设计规范（C端）
- 遵循现代后台管理系统设计规范（B端）
