# 项目信息

## 项目概述

这是一个游乐行业 SaaS 产品的高保真前端原型系统，包含 C 端小程序和 B 端管理后台两个部分。

## 项目背景

- **行业**: 游乐行业 SaaS
- **目标用户**: 
  - C端：游乐场顾客
  - B端：游乐场商家
- **项目性质**: 可交互的高保真前端原型
- **部署方式**: GitHub Pages

## 核心功能

### C 端小程序（顾客端）

提供给游乐场顾客使用的移动端应用，主要功能包括：

1. **会员系统**
   - 个人信息管理
   - 会员等级展示
   - 成长值系统

2. **票务管理**
   - 计次票统计
   - 计时票统计
   - 期限票统计

3. **营销功能**
   - 优惠券管理
   - 抽奖活动
   - 拼团功能
   - 秒杀活动
   - 砍价活动

4. **消费功能**
   - 储值管理
   - 订单查询
   - 发票开具

5. **社交分销**
   - 分享券
   - 二级分销
   - 拉新有奖
   - 发红包

6. **其他功能**
   - 实体卡绑定
   - 联盟商家
   - 自营商城
   - 关于我们

### B 端管理后台（商家端）

提供给游乐场商家使用的 PC 端管理系统，主要模块包括：

1. **日常运营**
   - 运营看板
   - 数据概览

2. **会员管理**
   - 会员概况统计
   - 会员详细信息管理
   - 游客信息查询
   - 会员分级管理
   - 会员标签系统
   - 储值记账管理
   - 卡片管理
   - 等级设置
   - 活动配置

3. **商品管理**
   - 套餐管理
   - 门票管理
   - 商品库存

4. **营销管理**
   - 营销活动配置
   - 优惠券管理
   - 促销活动

5. **分销管理**
   - 分销商管理
   - 分销业绩统计

6. **系统设置**
   - 店铺设置
   - 权限管理
   - 设备管理

7. **数据报表**
   - 经营报表
   - 会员报表
   - 财务报表

### 设计特点

**C 端设计**:
- 采用微信小程序风格
- 橙色渐变主题（原设计为粉色）
- 移动端优先设计
- 卡片式布局
- 网格化功能入口

**B 端设计**:
- 专业的后台管理风格
- 蓝色系主题
- 左侧导航 + 顶部栏布局
- 数据卡片展示
- 多级菜单结构

### 响应式设计

- **C 端**: 
  - 目标设备：手机
  - 基准宽度：375px
  - 最大宽度：md (768px)
  
- **B 端**:
  - 目标设备：PC
  - 最小宽度：1280px
  - 最佳宽度：1440px+

## 开发规范

### 组件规范

1. 使用函数式组件 + Hooks
2. 使用 JSX 语法
3. Props 解构传参
4. 合理使用 useState, useEffect 等 Hooks

### 样式规范

1. 优先使用 Tailwind CSS 工具类
2. 响应式设计使用 Tailwind 断点
3. 自定义颜色配置在 tailwind.config.js
4. 避免内联样式，除非必要

### 代码规范

1. 文件名使用 PascalCase（组件）
2. 变量名使用 camelCase
3. 常量使用 UPPER_SNAKE_CASE
4. 组件一个文件一个导出

## 性能优化

1. **代码分割**: 使用路由级别懒加载（可扩展）
2. **资源优化**: CDN 加载图标库
3. **构建优化**: Vite 自动优化打包
4. **图片优化**: 使用矢量图标，减少图片使用
